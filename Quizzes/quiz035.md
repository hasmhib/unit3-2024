## Code

```py
def count(list1: list, list2: list) -> list:
    output = []
    for x in list1:
        if x in list2:
            output.append(x)
    return output

list1 = [1, 2, 3, 4]
list2 = [3, 4, 5, 6]
print(count(list1, list2))
```
